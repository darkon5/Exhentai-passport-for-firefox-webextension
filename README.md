# Exhentai Passport

This is a webextension for firefox browser.

Say goodbye to sad panda.

## Requirement:
  firefox 48+
  
## Steps:

* Install this addon.
* Open exhentai.org. Click the sad panda image if you can see.
* Follow the hint and login the e-forums.
* Reopen exhentai.org.
* Enjoy :) .
